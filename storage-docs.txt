Storage API Documentation
Use these endpoints to interact with your storage buckets programmatically

API Authorization

All requests require the following authorization header:

Authorization: Bearer bytecloud_r8devs_r3g10n-1n-typegptxr8devsonskyflare
1. List Bucket Contents
Retrieve all files stored in a specific bucket.

curl -X POST "https://storageapis.skyflare.sh/v2/list-content" \
     -H "Authorization: Bearer bytecloud_r8devs_r3g10n-1n-typegptxr8devsonskyflare" \
     -H "Content-Type: application/json" \
     -d '{
           "folder_name": "YOUR_BUCKET_ID_HERE"
         }'

Example Response:

{
  "message": "Files retrieved successfully",
  "folder": "febb5fe3-97e7-45e4-91f8-2e1d0b8f5d5c",
  "file_count": 2,
  "files": [
    {
      "file_name": "image.png",
      "size": 1024,
      "uploaded_at": "2023-10-27T10:00:00.000Z",
      "access_link": "https://cdn.skyflare.sh/febb5fe3.../image.png"
    },
    {
      "file_name": "document.pdf",
      "size": 2048,
      "uploaded_at": "2023-10-27T10:05:00.000Z",
      "access_link": "https://cdn.skyflare.sh/febb5fe3.../document.pdf"
    }
  ]
}

2. Upload File
Upload a file to a specific bucket using multipart form data.

curl -X POST "https://storageapis.skyflare.sh/v2/upload-content" \
     -H "Authorization: Bearer bytecloud_r8devs_r3g10n-1n-typegptxr8devsonskyflare" \
     -F "folder_name=BUCKET_ID_HERE" \
     -F "file=@/path/to/image.png"

Example Response:

{
  "message": "Uploaded successfully",
  "file_name": "image.png",
  "file_link": "https://byclcdn.skyflare.sh/febb5fe3.../image.png"
}

3. Delete File
Delete a specific file from a bucket using its access link.

curl -X POST "https://storageapis.skyflare.sh/v2/delete-content" \
     -H "Authorization: Bearer bytecloud_r8devs_r3g10n-1n-typegptxr8devsonskyflare" \
     -H "Content-Type: application/json" \
     -d '{
           "folder_name": "BUCKET_ID_HERE",
           "file_link": "FILE_LINK_HERE"
         }'

Example Response:

{
  "status": "deleted",
  "message": "File deleted successfully",
  "file_link": "https://byclcdn.skyflare.sh/xyzabcd/image.png"
}
